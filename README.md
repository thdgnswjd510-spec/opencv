# 첫번째 실습과제: 그림판 만들기

1. 소개 
    - opencv 라이브러리를 사용한 간단한 그림판 

2. 주요 기능
    - 마우스 왼쪽 버튼 누른 채로 드래그 시 그리기 기능
    - 키보드 'c' 조작 시 화면 초기화
    - 키보드 'q' 또는 'esc' 조작 시 그림판 종료

3. 기술 스택
    -파이썬
    -opencv

4. 실행 방법
    - 코드 복사 후 실행

# 두번째 실습과제: 크로마키 합성

1. 소개

3. 기술 스택
    -파이썬
    -opencv

4. 실행 방법
   - 코드 복사 후 실행
